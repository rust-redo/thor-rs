# thor-rs

A super fast cli tool to find potentially problematic installed packages.

## Features

- [ ] Find outdated packages
- [ ] Find vulnerable packages
- [ ] Find missed packages
- [ ] Support workspace mode
  - [ ] npm
  - [ ] pnpm
  - [ ] yarn

## Install

```shell
$ npm i -g thor-rs
```
